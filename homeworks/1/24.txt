Напишите программу для сбора информации о книгах памяти с сайта https://bessmertnybarak.ru. Используйте эту программу для сбора информации о всех книгах памяти Российской федерации. Выложите на github код и ссылку на google drive с датасетом. 

Входные данные: 
ссылка на список книг https://bessmertnybarak.ru/books/

Выходные данные:
Excel-файл со следующими колонками:
1. Название книги;
2. Ссылка на книгу;
3. Ссылка на алфавитный список;
4. Описание книги.

Если на странице списка отсутствует какая-то информация из требуемых, в таблицу следует записать NA.

Используйте библиотеку BeautifulSoup (https://www.crummy.com/software/BeautifulSoup/bs4/doc/) для сбора информации с сайта и pandas (https://pandas.pydata.org/) для сохранения информации в excel-файл.
